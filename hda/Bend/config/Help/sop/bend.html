Bend SOP

Overview

This sop bends objects. The bend is more uniform than the twist sop. For instance if you bend a tube by 360 degrees it will form a torus. It also allows limits to be placed on the bend.

Usage

Input 1 is the polygonal geometry to be bent.

Parameters

Upper Limit Upper limit on bend [0-1] specified relative to the bounding box.
Pivot Bias Pivot point [0-1] specified relative to the bounding box.
Lower Limit Lower limit on bend [0-1] specified relative to the bounding box.
Scale Bend Scales the radius of the bend.
Bend Angle Amount of bend in degrees. [0-360] You can't use negative values, instead rotate the bend by 180 degrees to achieve the same thing.
Bend Axis The axis around which to bend.
Rotate Bend Rotate the bend around.
